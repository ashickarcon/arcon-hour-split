# Arcon Hour Split

Android app to calculate hour splits across multiple days.

## Features
- Calculate hour distribution across days
- Support for two different hour options
- Auto-search mode for optimal hour combinations
- Clean, mobile-friendly interface

## Build APK

This repository uses GitHub Actions to automatically build the APK.

### Steps:
1. Push this repository to GitHub
2. Go to **Actions** tab
3. Click **Build Android APK** workflow
4. Click **Run workflow**
5. Download APK from **Artifacts** once complete

## Install on Android
1. Download the APK from GitHub Actions artifacts or Releases
2. Enable "Install from Unknown Sources" in Android Settings
3. Tap APK file to install
4. Open "Arcon Hour Split" app

## Local Development

Test locally with:
```bash
pip install kivy
python main.py
```
